## 아이디어 A: "Emerald Prestige" — 에메랄드 프레스티지

**Design Movement**: Contemporary Luxury Editorial (현대 럭셔리 에디토리얼)

**Core Principles**:
1. 여백을 적극 활용한 숨쉬는 레이아웃 — 공간 자체가 고급스러움을 말한다
2. 강렬한 타이포그래피 계층 구조 — 제목은 크고 대담하게, 본문은 섬세하게
3. 이미지 중심 스토리텔링 — 각 목적지를 시각적 여행으로 표현
4. 색상의 절제된 사용 — 보라색과 녹색을 포인트로, 흰색을 기반으로

**Color Philosophy**:
- 배경: 순백색 (#FFFFFF) — 깨끗함과 신뢰감
- 주요 포인트: 딥 포레스트 그린 (#1B5E20) — 골프 코스의 자연
- 보조 포인트: 로열 퍼플 (#6A1B9A) — 품격과 고급스러움
- 액션 컬러: 크림슨 레드(#C62828) — 긴급함과 특가 강조
- 텍스트: 차콜 (#1C1C1E) — 가독성

**Layout Paradigm**:
- 비대칭 그리드: 큰 이미지 + 텍스트 블록이 교차하는 잡지형 레이아웃
- 풀스크린 히어로 슬라이더 (각 목적지별 배경)
- 패키지 카드: 수평 스크롤 캐러셀 방식
- 사이드 네비게이션 플로팅 바

**Signature Elements**:
1. 골프공 궤적 애니메이션 — 섹션 전환 시 골프공이 날아가는 효과
2. 목적지 카드에 국기 아이콘 + 그린 오버레이 호버 효과
3. 가격 태그에 빨간색 리본 배지

**Interaction Philosophy**:
- 스크롤 시 패럴랙스 효과로 깊이감 연출
- 호버 시 이미지 확대 + 오버레이 텍스트 등장
- 부드러운 페이드인 애니메이션

**Animation**:
- 히어로 섹션: 슬로우 줌인 배경 + 텍스트 슬라이드업
- 카드 호버: scale(1.03) + 그린 오버레이 fadeIn
- 섹션 진입: staggered fadeInUp (0.1s 간격)

**Typography System**:
- 제목: Playfair Display (세리프, 고급스러운 느낌)
- 부제목: Noto Sans KR Bold
- 본문: Noto Sans KR Regular
- 가격/숫자: DM Mono (모노스페이스, 명확함)

## 아이디어 C: "Verdant Journey" — 버던트 저니 ✅ 선택

**Design Movement**: Refined Contemporary Travel (세련된 현대 여행 미학)

**Core Principles**:
1. 자연의 풍요로움을 담은 색채 — 골프 코스의 초록빛을 브랜드 아이덴티티로
2. 비대칭 레이아웃으로 독창성 — 일반적인 격자형을 탈피한 유기적 구성
3. 이미지와 텍스트의 균형 — 시각적 임팩트와 정보 전달의 조화
4. 고급스러운 디테일 — 미세한 그림자, 그라디언트, 텍스처로 깊이감

**Color Philosophy**:
- 기본 배경: 순백색 (#FFFFFF) — 청결함과 신뢰
- 프라이머리: 딥 에메랄드 그린 (#0D5C3A) — 자연, 골프, 신뢰
- 세컨더리: 리치 퍼플 (#5B2D8E) — 품격, 고급스러움
- 액센트: 버밀리온 레드 (#E53935) — 특가, 긴급, 이벤트
- 서브 배경: 크림 (#F8F6F0) — 따뜻한 고급감
- 텍스트: 딥 차콜 (#1A1A2E)

**Layout Paradigm**:
- 풀스크린 히어로 + 오버레이 검색/CTA
- 비대칭 목적지 쇼케이스 (큰 이미지 + 작은 이미지 모자이크)
- 패키지 카드: 호버 시 상세 정보 슬라이드업

- 고정 사이드 플로팅 버튼 (카카오톡)

**Signature Elements**:
1. 섹션 구분선: 물결 모양 SVG 디바이더 (녹색 → 흰색)
2. 패키지 카드: 상단 국가 플래그 스트라이프 + 가격 배지
3. 히어로 텍스트: 금색 언더라인 강조

**Interaction Philosophy**:
- 스크롤 기반 애니메이션으로 스토리텔링
- 목적지 카드 호버: 이미지 줌 + 그린 오버레이 + 가격 표시
- 부드럽고 우아한 전환 — 빠르지 않고 여유있는 타이밍

**Animation**:
- 히어로: 배경 이미지 슬로우 줌 + 텍스트 fadeInUp (0.8s ease)
- 카드 호버: transform scale(1.04) + shadow elevation
- 섹션 진입: IntersectionObserver 기반 stagger fadeInUp
- 숫자 통계: countUp 애니메이션

**Typography System**:
- 한국어 제목: Noto Serif KR (세리프, 고급스러운 한국어)
- 영문 제목: Cormorant Garamond (우아한 세리프)
- 본문/UI: Noto Sans KR (가독성 최우선)
