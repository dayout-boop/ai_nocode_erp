## 5. OCR 엔진별 설치 가능성 분석

### 5.1 Tesseract OCR (권장)

| 항목 | 내용 |
|---|---|
| **apt 패키지** | `tesseract-ocr` v5.3.4 (설치 가능) |
| **한국어 데이터** | `tesseract-ocr-kor` v4.1.0 (설치 가능) |
| **Python 바인딩** | `pytesseract` v0.3.13 (pip 설치 가능, 의존성 1개) |
| **설치 용량** | 약 50~100 MB (경량) |
| **RAM 요구량** | 약 100~300 MB |
| **처리 속도** | CPU에서 이미지 1장당 0.5~3초 |
| **설치 가능 여부** | ✅ **가능** |

```bash
# 설치 명령
sudo apt-get install -y tesseract-ocr tesseract-ocr-kor
sudo pip3 install pytesseract
```

### 5.2 EasyOCR

| 항목 | 내용 |
|---|---|
| **pip 패키지** | `easyocr` v1.7.2 (설치 가능) |
| **의존성** | PyTorch 2.12.0 (532 MB), CUDA 관련 패키지 포함 총 **38개 패키지** |
| **설치 용량** | PyTorch 단독 532 MB + 기타 → **총 2~3 GB 예상** |
| **RAM 요구량** | CPU 모드 최소 2~4 GB (현재 가용 1.8 GB로 **부족**) |
| **처리 속도** | CPU에서 이미지 1장당 10~60초 |
| **설치 가능 여부** | ⚠️ **조건부 가능** (디스크 여유 있으나 RAM 부족 위험) |

### 5.3 Google Cloud Vision API / Gemini Vision (현재 연동 가능)

| 항목 | 내용 |

## 6. 종합 권고사항

### 단기 권고 (즉시 적용 가능)

**Gemini Vision API 활용**이 가장 현실적입니다. 이미 `GEMINI_API_KEY`가 서버에 주입되어 있고, 별도 설치 없이 한국어 예약 문서(카카오톡 캡처, 이메일 스크린샷, PDF 등)를 처리할 수 있습니다.

### 중기 권고 (설치 후 사용)

**Tesseract OCR**은 경량(~100 MB)이고 RAM 부담이 적어 현재 서버 환경에서 안정적으로 운영 가능합니다. 단, 복잡한 레이아웃이나 손글씨 인식률은 Gemini Vision보다 낮습니다.

### EasyOCR 주의사항

현재 RAM(가용 1.8 GB)으로는 PyTorch 기반 EasyOCR 구동이 불안정할 수 있습니다. 설치는 가능하나 실제 추론 시 OOM(Out of Memory) 오류 발생 가능성이 있습니다.

## 7. 추천 구현 방향

두골프 ERP의 예약 자동화 목적에 맞는 OCR 구현 방향은 다음과 같습니다.

| 우선순위 | 방법 | 이유 |
|---|---|---|
| 1순위 | **Gemini Vision API** | 즉시 사용 가능, 한국어 최고 정확도, 비용 저렴 |
| 2순위 | **Tesseract + 전처리** | 오프라인 처리, 비용 없음, 설치 용이 |
| 3순위 | **EasyOCR** | RAM 업그레이드 후 고려 |
