# dogolf.com 상품 상세 페이지 UI 참고 구조

## 출발일 선택 방식
- 달력(Calendar) 형태로 월별 2개 나란히 표시
- 각 날짜 셀에 요금(예: 46.0만, 39.0만)이 표시됨
- 과거 날짜 및 예약 불가 날짜는 빨간색으로 표시 (비활성화)
- 오늘 이후 날짜만 선택 가능
- 날짜 클릭 시 하단 "행사 선택" 섹션에 해당 날짜의 행사 정보 표시

## 행사 선택 구조
- 날짜 선택 후 "행사 선택" 섹션에 카드 형태로 행사 표시
- 행사 카드: 예약가능/마감 뱃지, 상품명, 잔여인원(현재예약/최대예약), 최소출발인원, 상품금액(성인/아동/유아)
- "선택됨" 버튼으로 행사 선택
- 선택 후 인원 수 입력 (성인/아동/유아 각각 +/- 버튼)
- 최종 예약하기 버튼

## 요금 구조
- 날짜별로 요금이 다름 (요일/시즌별 차등)
- 확정요금: 26년 5월 30일 까지
- 미확정요금: 26년 5월 30일 이후
- 기준: 성인 1인 기준, 최저가 표시

## 상품 탭 구조
- 상품선택 / 주요 여행일정 / 요약정보 / 상품정보 / 여행일정 / 약관/환불규정

## 적용할 개선사항 (두골프 ERP/홈페이지)
1. 상품 상세 페이지에 달력 형태 날짜/요금 선택 UI 추가
2. 오늘 이후 날짜만 표시 (과거 날짜 비활성화)
3. 미래 출발일이 없는 상품 → 최저가 "개별문의" 표시 + 목록 우선순위 후순위
4. 날짜별 요금 입력은 ERP 상품 등록/수정에서 관리
5. 행사(출발일별 이벤트) 개념을 ERP 상품에 연결
