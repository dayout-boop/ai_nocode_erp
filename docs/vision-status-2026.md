# 두골프 AI 플랫폼 — 핵심 현황 및 비전 문서

**작성일**: 2026년 5월 5일  
**대상**: dogolf-tour-dkz3fsmp.manus.space (두골프 ERP + 홈페이지)

---

## 1. 나는 누구인가 (현재 상태)

**주식회사 투어커뮤니케이션** — 국내외 골프투어 전문 여행사 (브랜드명: 두골프, dayoutgolf.com)

현재 두골프 ERP + 홈페이지를 직접 AI와 함께 개발하며 운영 중이다.
단순한 여행사 ERP를 넘어, **AI 기반 여행사 플랫폼 SaaS**로 전환하는 것이 핵심 목표다.
두골프는 이 플랫폼의 첫 번째 샘플 사이트이자 실증 사례다.

---

## 2. 현재 구축된 것 (Phase 1~2 완료)

### 두골프 ERP 핵심 기능 (135개 이상)
- 상품관리 / 예약관리 / 정산관리 / CRM / CMS
- 수기 예약 + 견적서 자동 생성 (변수 치환 시스템)
- 이미지 업로드 + AI 자동생성 (Gemini + Pixabay)
- 구글 스프레드시트 데이터 10,000건 DB 이식 완료

### AI 시스템 3계층
| 인터페이스 | 대상 | 역할 |
|---|---|---|
| 두골프 마스터 (Master) | 관리자 | 전체 감독·개발 요청·분석 |
| 골프톡 (GolfTalk) | 고객 | 상품 문의·예약 상담 |
| 두골프 매니저 (Manager) | 파트너 | 파트너별 예약·정산 관리 |

### AI 엔진 현황
- OpenRouter 기반 복잡도별 모델 자동 라우팅 (high/medium/low)
- Tool Calling 10개 (예약/정산/상품/비용/오류로그 등 ERP 직접 조회)
- SSE 실시간 스트리밍 응답
- Manus API 개발 요청 자동 전송 파이프라인
- GitHub 연동 완료 (커밋 자동 연결 + 유사 코드 검색 + 중복 개발 방지)

### 외부 연동 완료
- Stripe 결제 / 포트원 V2 / 카카오 알림톡 (Solapi)
- Runway ML 동영상 자동생성
- n8n 자동화 파이프라인
- GitHub API (코드 히스토리 관리)

## 4. 비전 — 내가 만들고 싶은 것

### 단기 목표 (2026년 상반기)
1. **두골프 AI 플랫폼 완성** — Phase 3~4 구현으로 Manus 크레딧 50% 이상 절감
2. **파트너 온보딩 자동화** — 사업자등록증 OCR + 자동 승인 + 하위 담당자 관리
3. **partner.dayoutgolf.com** — 파트너 전용 랜딩페이지 + 매니저 챗봇 가입 플로우

### 중기 목표 (2026년 하반기)
1. **SaaS 플랫폼 런칭** — 두골프를 샘플로 타 여행사/골프장에 플랫폼 판매
2. **1,000개 파트너 지원 구조** — 파트너별 독립 AI 에이전트 + 테넌트 격리
3. **자기학습 피드백 루프** — AI 호출 결과 자동 평가 → 모델 선택 최적화

### 장기 비전
> "두골프 마스터 하나로 전체 여행사 운영을 자율 수행하는 AI 플랫폼"

- 관리자가 채팅 한 줄로 상품 등록 → 이미지 생성 → SNS 배포 → 예약 접수 → 정산까지 자동 처리
- 신규 파트너가 가입하면 자동으로 ERP + 홈페이지 + AI 챗봇이 세팅되는 구조
- Manus 의존도를 최소화하고 자체 AI 엔진으로 운영 비용 70% 절감

## 6. 현재 기술 스택

```
Frontend:  React 19 + Tailwind 4 + tRPC 11 (TypeScript)
Backend:   Express 4 + tRPC 11 + MySQL (Drizzle ORM)
AI:        OpenRouter (모델 라우팅) + Gemini API + Manus API
Storage:   S3 (이미지/파일)
Auth:      Manus OAuth (JWT 세션 쿠키)
Infra:     Manus WebDev (dogolf-tour-dkz3fsmp.manus.space)
Domains:   dayoutgolf.com / partner.dayoutgolf.com
```
